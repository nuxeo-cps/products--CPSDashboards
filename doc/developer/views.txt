RoadmapView
------------

This view class inherits form searchview.SearchView for users and
groups search forms. It also provides some stack related helpers.

ReuseAnswerView
---------------

This view class inherits from searchview.SearchView for existing
outgoing mails search/forms and results.

FolderLocalRolesView
--------------------

This view class inherits from searchview.SearchView. It recognizes
whether we search for users or groups::
